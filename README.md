# README

API demonstration to query a zip code.


TECHNOLOGY:

* Ruby version 2.6.3

* SQLite 3


INSTALL:

* git clone github.com/tuballeonel/cep_query

* cd cep_query

* bundle install

* rails db:create

* rails db:migrate

* rails db:seed


RUN THE PROJECT:

* rails server or rails s
